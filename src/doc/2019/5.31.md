#### mac截图

```
command + shift + 3 截取全屏
command + shift + 4 截取选定内容
```

1.打开一个窗口光标移动到终端，命令+ N代开一个新的窗口

2.在一个新窗口中建立多个终端窗口，命令+ T，即可实现

#### 终端使用

```
command + t 在当前终端打开一个新的窗口
command + n 打开新的终端
```
