## 手写Spring

### AOP

AOP主要应用场景：日志，事务，权限

AOP是面向切面，解决了代码复用的问题，AOP的核心是在方法之前或者方法之后处理一些事情。AOP的底层实现是通过代理模式。

















































































```java
@JsonGetter("Type")
@JsonSetter("Type")
@JsonProperty(value = "groupId")
```
